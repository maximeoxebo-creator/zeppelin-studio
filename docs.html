Installation

  1. In the Shopify admin, open Online Store → Themes.
  2. Click Add theme → Upload zip file and select the theme .zip.
  3. Once imported, click Publish to make it live, or Customize to edit it first.
The message “Updates are not supported for themes uploaded manually” is normal for any manually installed theme. It does not affect how the theme works.

Customization basics

Everything is configured in the Shopify theme editor — no code required.

Colors & typography

Under Theme settings, adjust color schemes, heading and body fonts, and the type scale. Zeppelin ships with Fraunces (headings) and DM Sans (body), self-hosted for speed.

Sections

Add, reorder, and configure sections from the editor: banners, featured collections, rich text, multicolumn, testimonials, lookbook, and many more. Every section has padding controls and mobile-specific behavior where relevant.

Header & footer

Logo, navigation menu, icons (search, account, cart), and footer blocks are set in their respective section groups.

The editor caches section schemas per session. After a theme update, fully close and reopen the editor (or hard-refresh with Cmd/Ctrl + Shift + R) so new settings appear.

Product color variants Advanced

The product page reacts live when a customer switches color, with no page reload. When set up correctly, changing the color swatch updates the gallery, the title area, and the price in place.

How it works

  • Color swatches — enable swatches in the main product section. Each variant’s color becomes a clickable chip instead of a dropdown.
  • Per-color media filtering — assign each product image to a color in Shopify (the image’s “Media by variant / color” association). When a color is selected, the gallery filters to show only that color’s media. Images with no color assigned always stay visible.
  • Live text update — the product title, eyebrow, and description update instantly when the variant changes, so color-specific copy stays in sync.

Setup checklist

  1. In the product admin, add a Color (or equivalent) option with your variants.
  2. Upload the product images, then use “Select media by variant” to attach each image to its color.
  3. In the theme editor, open the product section and enable color swatches and filter gallery by color.
  4. Preview the page and click through each swatch to confirm the gallery filters correctly.
If images don’t filter when switching color, the media isn’t associated with a variant in the product admin. Filtering is driven entirely by that association — the theme reads it automatically.

Alternate product pages Advanced

Zeppelin includes ready-made alternate product templates so different products can have a completely independent layout and content — each anchored to its own product.

Why use them

The default product template drives every product. When you want one product (or a group) to have its own eyebrow, description block, and collapsible tabs — without affecting the rest of the catalog — assign it an alternate template instead of editing the shared one.

Available templates

TemplateWhat it’s for
DefaultThe standard product layout, applied to every product unless overridden.
MinimalPared-back layout for simple products.
EditorialStory-driven layout with more room for narrative.
Product 1–10Ten independent templates, each pre-built with an eyebrow block, a description block, and four collapsible tabs (Materials, Shipping & Returns, Dimensions, Care).

How to assign a template to a product

  1. Open the product in Products in the Shopify admin.
  2. In the right sidebar, find the Theme template selector.
  3. Choose an alternate template (e.g. product.1, minimal, editorial) and save.
  4. Open that product in the theme editor — you’re now editing only that template. Its eyebrow, description, and tab content are unique to the products using it.
Each Product 1–10 template is fully independent. Editing one never changes another, so you can give up to ten products (or ten product families) their own bespoke page.

Glassmorphism (liquid-glass) sections Advanced

Zeppelin includes a family of frosted-glass sections — Glass Hero, Glass Cards, Glass Key figures, Glass Testimonial, and Glass CTA — plus an optional glass panel on the Scattered images section and a glass toggle on the Marquee. They layer a translucent, blurred panel over a background image or color.

Common settings

SettingWhat it does
Show panelToggles the frosted panel on or off. With it off, text sits directly on the background.
Panel opacityHow opaque the glass is. Lower = more of the background shows through; higher = more legible text.
Panel shapeRounded or square corners on the glass panel.
Background typeImage or solid color behind the glass.
Panel text colorGlobal color for all panel text.
Title / eyebrow / subtitle colorOptional per-element overrides on top of the global text color.

Getting a clean glass effect

  • Use a background image with enough contrast behind the panel — busy, high-contrast photos read best through frosted glass.
  • Keep panel opacity high enough that text stays legible (start around the mid-range and adjust).
  • The blur is applied automatically. For accessibility, the panel falls back to a near-solid background when a visitor’s device requests reduced transparency.
The global cookie banner also uses the liquid-glass style and can be toggled in Theme settings.

Scattered images Advanced

This section places images freely across the canvas with a subtle parallax on scroll, plus a centered headline. Each image is a block you position independently.

Per-image settingWhat it does
Position X / YPlaces the image on desktop, as a percentage of the canvas.
Position X / Y (mobile)Independent placement for mobile, so images don’t overlap on small screens.
WidthImage size (rem). On mobile it’s additionally capped relative to the viewport.
ParallaxHow much the image drifts on scroll.

An optional frosted-glass panel can sit behind the centered text (with its own opacity and shape controls) — see Glassmorphism sections.

Set the desktop positions first, then switch the editor preview to mobile and set the mobile X/Y for each image — the two are independent, so a mobile layout won’t disturb your desktop composition.

Mobile text & button sizes

Sections like the Slideshow, Image/video banner, Categories, and Marquee expose separate mobile size controls for headings, body text, and buttons.

Mobile sizes work as a multiplier on top of the desktop size: 100% means “same as desktop,” higher enlarges, lower shrinks — for mobile only. The desktop size always applies first, so you only touch the mobile control when small screens need a tweak.

Title vs. body colors

Sections with both a heading and running text — such as Before/After, Lookbook, and Countdown — have separate color settings:

  • Title color applies only to the heading.
  • Body / text color applies only to the running text.
  • Eyebrow color is independent and defaults to your accent color.

Leaving a color empty keeps the theme’s default for that element.

Best practices

  • Use images at the right ratio (portrait 3:4 for products) for crisp rendering.
  • Enable scroll-reveal animations for a subtle editorial feel.
  • Always test on mobile after changing a layout.
  • Your saved content is safe across theme updates — settings and uploaded images persist.
